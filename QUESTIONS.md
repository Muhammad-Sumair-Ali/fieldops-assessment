## QUESTIONS

These are the questions I would have asked before starting:

1. Should clients be able to create jobs themselves, or only admins?
2. What happens to assigned jobs if a technician is removed from the system?
3. Should job reassignment be restricted by status (e.g., cannot reassign a completed job)?
4. Is there a need for multiple admins, or is it always a single admin account?
5. Should cancelled jobs be recoverable, or permanently closed?
6. Are there any SLA requirements for job completion?
